This is an open source application licenced under the GPL v2.

It uses the following libraries:

  • StickyListHeaders, to make sure the date is always visible when you scroll the Line-up section
  • Typeface TextView, to display custom fonts
  • Android NumberPicker, to support NumberPicker on older devices
  • Google Play Services Library, to display the map
  • Paypal Mobile SDK, to allow you to donate
It also uses some classes written by Laurent Michenaud and the NASA.
